# DashboardEditor (MVP)

This is a minimal working editor:
- Screens + objects list (z-order up/down)
- Canvas view with drag (LMB)
- Inspector (Name/Transform/Gauge)
- Assets modal (upload/select/delete for Images/Fonts)
- Import/Export `.dashboard` (custom header + zip offset)

## Requirements
- Node.js LTS (18+ recommended) + npm

## Run
```powershell
cd .\dashboard-editor
npm install
npm run dev
```

## Build
```powershell
npm run build
npm run preview
```
